    * If you are hosting your own Docker container, the username matches the USER environment variable (under the "Settings" tab in Kitematic). The password is the same as the username. In addition, any path on the host may be mounted as volume "/share" on the container. Note that doing so requires starting a new container. From the Kitematic "Settings" tab for the current container, select "Volumes" and click "CHANGE" in the row for "/share". Choose, or create, a folder, in your filesystem.
